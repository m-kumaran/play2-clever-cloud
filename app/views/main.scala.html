This is a sample Play 2.1.1 Application with PostgreSQL deployed in Clever-Cloud

Steps to deploy:-

  • Create an account in Clever-Cloud with just email and password
  • Add a new application and configure according to your requirement and note down the value of <YOUR_GIT_DEPLOYMENT_URL>
  • Add a new postgresql service under Services menu.. After a while you will get an email with the following details:-
    • db_host = <YOUR_UNIQUE_DBNAME>.postgresql.clvrcld.net
    • db_port = 5432
    • db_name = <YOUR_UNIQUE_DBNAME>
    • db_username = <YOUR_USERNAME>
    • db_password = <YOUR_PASSWORD>
  • Amend your application.conf accordingly (The url value should be inside double quotes)
    db.default.driver=org.postgresql.Driver
    db.default.url="jdbc:postgresql://<YOUR_UNIQUE_DBNAME>.postgresql.clvrcld.net:5432/<YOUR_UNIQUE_DBNAME>"
    db.default.user=<YOUR_USERNAME>
    db.default.pass=<YOUR_PASSWORD>
  • In the project root folder create a folder with the name "clevercloud" and put a file inside that folder, with the name "play.json"
    If you use play 2.1.1 and have a dedicated config file for this cloud provider, then the play.json file shall contain
    { "deploy": { "goal": "-Dplay.version=2.1.1 -Dconfig.resource=application-cc.conf" } }
  • Issue the following git commands from the project root. [If you don't have GIT download from http://git-scm.com/downloads
    git init
    git remote add clevercloud <YOUR_GIT_DEPLOYMENT_URL> (clevercloud is the sample name given for this remote. You can change it if you need)
    git push clevercloud master
    You will see the output like :-
    C:\your_project_root>git push clevercloud master
    Counting objects: 7, done.
    Delta compression using up to 8 threads.
    Compressing objects: 100% (3/3), done.
    Writing objects: 100% (4/4), 328 bytes, done.
    Total 4 (delta 2), reused 0 (delta 0) remote: {"id":999,"message":"The application has successfully been queued for redeploy.","type":"success"}
    To <YOUR_GIT_DEPLOYMENT_URL> XXX..XXX master -> master
  • Thats it!!! Follow the logs from the clever-cloud LOGS menu
